## How to install

- `npm i -g adeng-jenkins-cli`

## How to use

- `jenkins -h` <- it will give you list of commands

## Development Guide

- `npm ci`
- `npm link` - it will link package so it will be available globally
- `npm run watch` - it will compile files to dist directory
